# انٹرایکٹو کویری سسٹم

## جائزہ
انٹرایکٹو کویری سسٹم صارفین کو جسمانی مصنوعی ذہانت اور ہیومنوائڈ روبوٹکس کتاب کے مواد کے بارے میں سوالات پوچھنے اور مناسب حوالہ جات کے ساتھ AI تیار کردہ جوابات حاصل کرنے کی اجازت دیتا ہے۔ سسٹم صفر ہلیو سینیشن پالیسی کے ساتھ آئینی مطابقت برقرار رکھتے ہوئے ایک بے داغ صارف تجربہ فراہم کرتا ہے۔

## خصوصیات
- **کویری ان پٹ**: صارفین کتاب کے مواد کے بارے میں سوالات جمع کروا سکتے ہیں
- **AI جوابات**: کتاب کے مواد کے مطابق صفر ہلیو سینیشن کے ساتھ تیار کردہ جوابات
- **حوالہ ٹریکنگ**: تمام جوابات مناسب حوالہ جات کے ساتھ شامل ہیں جو ماخذ مواد دکھاتے ہیں
- **خرابی کا انتظام**: مختلف خرابی کی حالت کے لیے واضح فیڈ بیک
- **شرح حد**: API کے غلط استعمال سے بچاؤ

## استعمال کیسے کریں
1. `/query` صفحہ پر جائیں
2. ٹیکسٹ ایریا میں اپنا سوال درج کریں
3. "کویری جمع کرائیں" پر کلک کریں
4. حوالہ جات کے ساتھ AI تیار کردہ جواب دیکھیں
5. ماخذ مواد دیکھنے کے لیے حوالہ جات کے لنکس پر کلک کریں

## تکنیکی تفصیلات
- **فرنٹ اینڈ**: ڈوکوسورس کے ساتھ یکجا کردہ ری ایکٹ کمپونینٹس
- **بیک اینڈ**: فیز 3 جواب تیار کرنے والا سسٹم
- **API**: `/api/answer` کویری کی پروسیسنگ کے لیے اختتامی نقطہ
- **حوالہ جات**: شفافیت کے لیے ماخذ یو آر ایلز اور چنک آئی ڈیز

## جواب کی اقسام
- **ANSWERED**: حوالہ جات کے ساتھ درست جواب
- **INSUFFICIENT_CONTEXT**: دستیاب مواد کے ساتھ کویری کا جواب نہیں دیا جا سکتا
- **REFUSED**: آئینی مطابقت کے قوانین کے مطابق کویری سے انکار کر دیا گیا

## سیکیورٹی
- ان پٹ کی توثیق اور صفائی
- XSS کی روک تھام
- API کے غلط استعمال کو روکنے کے لیے شرح حد
- فرنٹ اینڈ پر کوئی API کلیدیں نہیں دکھائی گئیں

## رسائی
- مناسب ARIA لیبلز
- کی بورڈ نیویگیشن کی حمایت
- اسکرین ریڈر کے دوست
- پڑھنے کے لیے زیادہ کانٹراسٹ
